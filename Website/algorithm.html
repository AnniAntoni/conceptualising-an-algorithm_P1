Our Algorithm

Deciding what cereals to buy can be a nightmare if you don't know where to start. Our solution will make it easier to decide by narrowing down the choices. The algorithm provides a curated list of recommended cereals according to user preferences.

Attributes

There are four main attributes in our algorithm based on the dataset we curated:

  • type of cereal, string, "hot" or "cold" Initial user preference of whether the user prefers hot cereal or cold cereal for breakfast
  • calories per serving, integer, 50 Defines how important calory intake is for the user, can help in finding low calory options if it is important
  • sugar content, float, 6.7 Is it important for the user that the cereals are sweet? If not let's suggest healthier options.
